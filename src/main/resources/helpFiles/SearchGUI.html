SearchGUI

graphical user interface for proteomics identification search engines

SearchGUI is a user-friendly open-source graphical user interface for configuring and running proteomics identification search engines and de novo sequencing algorithms.

For more details and help see the SearchGUI home page: http://compomics.github.io/projects/searchgui.html.

To visualize and analyze SearchGUI results we recommend the use of PeptideShaker.

If you find any bugs or have any questions please contact the developers.

Please cite SearchGUI as Barsnes and Vaudel: J Proteome Res. 2018 Jul 6;17(7):2552-5.
