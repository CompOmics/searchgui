Introduction

SearchGUI allows you to inspect implemented modifications as well as manage user modifications in a user friendly way. If you encounter any difficulty with modifications in SearchGUI, please contact the developers.

Default Modifications

The default modifications are basically the ones implemented in the OMSSA search engine. Note that all modifications are not compatible with X!Tandem and can result in nonsense matches. For instance, X!Tandem does not support modification at specific amino acids on peptide termini.

User Modifications

User modifications can be implemented very easily in SearchGUI. Note that user modifications will be available in other instances of SearchGUI and PeptideShaker.
